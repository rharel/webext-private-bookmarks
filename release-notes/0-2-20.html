Private Bookmarks 0.2.20

These are the release notes for Private Bookmarks 0.2.20, released on 12/06/2022.

What's new
  • This release should provide stability improvements and bug fixes.
  • New locales: German (de) and Italian (it)
  • The experimental sync feature has been removed. This due to Firefox now imposing a strict limit on the size of data extensions are allowed to sync. That limit is too small to support the sync feature for all users.

Something's not working? Got a question and/or suggestion? Post it here.

As always with a new release, be on the lookout for bugs and oversights, and make backups frequently!

If you do not wish to be notified of new features after an update, you can un-subscribe through the extension's options page.
